Apa Itu Solidity?

  • Solidity adalah bahasa tingkat tinggi yang berorientasi objek untuk mengimplementasikan smart contract. Bahasa ini dirancang untuk menargetkan Ethereum Virtual Machine (EVM).

  • Bahasa ini diketik secara statis, mendukung pewarisan, pustaka, dan tipe yang ditentukan pengguna yang kompleks di antara fitur-fitur lainnya.

Contoh smart contracts di solidity:

// SPDX-License-Identifier: GPL-3.0
pragma solidity >= 0.7.0;
                
contract Coin {

// Variabel publik untuk menyimpan alamat pembuat koin
address public minter;

// Mapping untuk menyimpan saldo koin per alamat
mapping (address => uint) public balances;
                
// Aksi yang memungkinkan klien bereaksi pada perubahan kontrak
event Sent(address from, address to, uint amount);
                
// Konstruktor yang hanya dijalankan saat kontrak dibuat
constructor() {
minter = msg.sender; // Menetapkan pembuat kontrak sebagai pembuat koin
}
                
// Fungsi untuk menciptakan koin baru dan mengirimnya ke alamat
function mint(address receiver, uint amount) public {
require(msg.sender == minter, "Hanya pembuat koin yang bisa menciptakan koin baru.");
require(amount < 1e60, "Jumlah koin terlalu besar.");
balances[receiver] += amount;
}
                
// Fungsi untuk mentransfer koin
function send(address receiver, uint amount) public {
require(amount <= balances[msg.sender], "Saldo tidak cukup.");
balances[msg.sender] -= amount;
balances[receiver] += amount;
emit Sent(msg.sender, receiver, amount);
    }
}

Variabel dan tipe:

  • Local

Dideklarasikan di dalam fungsi dan tidak disimpan on-chain

  • State

Dideklarasikan di luar fungsi untuk mempertahankan keadaan global dalam kontrak. dan disimpan di dalam on-chain

  • Global

Memberikan informasi tentang rantai. Informasi ini diinjeksikan oleh EVM selama proses berjalan dan tidak dideklarasikan oleh pembuat kontrak. Termasuk hal-hal seperti pengirim transaksi, hash blok, stempel waktu, dll.
